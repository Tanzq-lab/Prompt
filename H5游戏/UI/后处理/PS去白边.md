# Core Principle

去白边不是只有一种方法。

请先判断白边类型，再选择处理策略：

1. 如果白边是“没抠干净的背景残留”，优先用选区收缩、最小值、蒙版收缩处理。
2. 如果白边是“半透明像素混入白色背景”，优先用净化颜色、边缘颜色去污染处理。
3. 如果白边出现在头发丝、毛发等细节区域，优先用选择并遮住、调整边缘画笔、净化颜色、加深工具或局部蒙版手修。
4. 如果白边很少，优先用蒙版手修，不要全图强处理。
5. 如果主体本身是白色，不要把主体边缘误删；此时要根据背景色判断是否真的是边缘污染。
6. 如果最终使用场景是白底，白边不明显时可以不处理；如果用于游戏透明贴图、深色背景、彩色背景，则必须检查并清理。

在正式处理前，必须做边缘检查。

请使用以下预览背景检查，但不要合并到最终图片：

1. 透明棋盘格背景：检查 Alpha 基础状态
2. 红色背景：检查白边和浅色污染
3. 蓝色背景：检查白边和浅色污染
4. 黑色背景：检查白边、灰边、半透明发白
5. 白色背景：判断白边是否只在白底使用场景中不可见

检查结论需要写入报告。

## 3. 判断问题类型

请将每张图片的问题归类为以下类型之一或多个：

1. 无明显问题：边缘干净，不需要处理
2. 明显白边：边缘有连续白色或浅色轮廓
3. 半透明白污染：透明边缘混入白底颜色，深色背景下发白
4. 灰边 / 脏边：边缘残留灰色或脏色
5. 黑边：浅色背景下边缘发黑
6. 抠图毛刺：边缘有碎点、锯齿、残留像素
7. 毛发复杂边缘：头发丝、毛发、羽毛、毛绒等细节区域有白边
8. 主体白色风险：主体本身有白色区域，容易误删
9. 描边风险：主体本身有黑色线稿或描边，不能误删
10. 不建议自动处理：强处理会破坏主体细节

## 4. 选择处理方案

请根据问题类型选择合适方案，不要所有图片都套同一种方法。

---

### 方案 A：选区收缩 1px 到 2px

适合：

1. 明显白边
2. 边缘比较硬的物体
3. UI 图标、道具、普通角色边缘
4. 不涉及头发丝、毛发、透明阴影的素材

处理逻辑：

1. 基于主体透明区域建立选区。
2. 将选区向内收缩 1px。
3. 如果 1px 不够，再尝试 2px。
4. 使用蒙版控制，不要直接破坏原图。
5. 检查主体是否变瘦、尖角是否丢失、细节是否被吃掉。
6. 如果损伤明显，撤回并换用更保守方案。

禁止：

1. 不要默认收缩超过 2px。
2. 不要对毛发、头发丝、半透明阴影直接粗暴收缩。
3. 不要把主体高光当成白边抠除。

---

### 方案 B：选择并遮住 + 调整边缘

适合：

1. 头发丝
2. 毛发
3. 羽毛
4. 半透明边缘
5. 边缘复杂的角色
6. 抠图边缘不稳定的素材

处理逻辑：

1. 载入主体选区。
2. 进入“选择并遮住”。
3. 使用调整边缘画笔处理白边区域。
4. 重点检查边缘检测、平滑、羽化、对比度、移动边缘。
5. 需要时开启“净化颜色”。
6. 输出为带图层蒙版的新图层或保留透明 PNG。
7. 处理后在红 / 蓝 / 黑背景下复查。

建议参数范围：

1. 平滑：0 到 3
2. 羽化：0 到 0.5px
3. 对比度：0% 到 15%
4. 移动边缘：-5% 到 -20%
5. 净化颜色：根据白边程度逐步增加，不要一次拉满

禁止：

1. 不要为了去白边牺牲头发丝。
2. 不要把边缘处理成硬切。
3. 不要让角色边缘出现明显糊边。

---

### 方案 C：净化颜色 / 去除半透明白污染

适合：

1. 原图来自白底抠图
2. 半透明边缘混入白色
3. 深色背景下边缘发白
4. 选区边缘看似透明但 RGB 被白色污染

处理逻辑：

1. 识别 Alpha 边缘半透明像素。
2. 判断这些像素是否明显偏白、偏灰、偏原背景色。
3. 将边缘颜色向主体附近颜色靠拢。
4. 保留 Alpha 透明层级。
5. 不直接删除这些半透明像素。

禁止：

1. 不要把所有边缘统一变暗。
2. 不要污染主体原有颜色。
3. 不要把白色主体误判成白边。

---

### 方案 D：最小值处理蒙版

适合：

1. 已经有蒙版的抠图素材
2. 白边比较均匀
3. 需要让蒙版向内收一点
4. 边缘不是特别复杂的图片

处理逻辑：

1. 基于主体创建或使用已有蒙版。
2. 对蒙版使用“滤镜 > 其他 > 最小值”。
3. 半径从 0.5px 或 1px 开始。
4. 只在必要时增加半径。
5. 处理后检查轮廓是否变瘦。
6. 如果主体细节被吃掉，撤回并换更保守方案。

建议：

1. 普通角色 / 道具：优先 1px
2. 白边较明显：可测试 2px
3. 复杂毛发：慎用或不用

禁止：

1. 不要对所有图片统一套最小值。
2. 不要用过大半径。
3. 不要破坏尖角、发丝、手指、衣服褶皱边缘。

---

### 方案 E：蒙版手修

适合：

1. 白边只出现在局部
2. 自动处理会误伤主体
3. 边缘有复杂细节
4. 少量残留需要精准修掉

处理逻辑：

1. 保留图层蒙版。
2. 在蒙版上用黑色画笔隐藏白边区域。
3. 用白色画笔恢复误删区域。
4. 使用柔边画笔，让过渡自然。
5. 只处理问题区域，不做全图破坏性操作。

禁止：

1. 不要直接橡皮擦删除原图。
2. 不要让边缘出现硬切。
3. 不要把主体细节涂掉。

---

### 方案 F：加深工具压暗白边

适合：

1. 毛发、头发丝、羽毛等细节边缘
2. 想保留细碎边缘，但白边太亮
3. 白边不适合直接抠除
4. 需要把白边自然转成更接近主体的暗色边缘

处理逻辑：

1. 使用加深工具。
2. 选择柔边圆画笔。
3. 降低强度，多次轻涂。
4. 沿着白边区域自然过渡。
5. 不要只涂一个局部，避免出现一块深一块浅。
6. 处理后检查整体边缘是否自然。

禁止：

1. 不要把边缘压得过黑。
2. 不要破坏毛发层次。
3. 不要让角色边缘出现脏黑描边。
4. 不要用于本身是浅色主体的边缘，除非确认是白边污染。

## 5. 处理优先级

请按以下优先级选择方案：

1. 先检查白边是否真的影响使用场景。
2. 如果只用于白底且白边不可见，可不处理。
3. 如果是普通硬边素材，优先选区收缩或最小值。
4. 如果是半透明污染，优先净化颜色。
5. 如果是头发丝、毛发、羽毛，优先选择并遮住或加深工具。
6. 如果只有局部残留，优先蒙版手修。
7. 如果自动处理风险高，标记为建议手动精修。
